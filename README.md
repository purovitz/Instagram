# Instagram


## What is it?

It is a simple instagram clone built using PHP 8  + Symfony 6, MYSQL and HTML + CSS + Docker.

## How it works

The application can be divided into 4 users:

* Guest
  * Can register
  * It can reset its password
  * It can log on
* Registered user
  * Can register
* Moderator
* Admin
